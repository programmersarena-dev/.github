# ProgrammersArena

Welcome to ProgrammersArena, an online judge platform designed for coding contests, problem solving, and competitive programming practice.

## About

ProgrammersArena provides a complete environment for:
- submitting code in multiple languages
- evaluating submissions against test cases
- tracking rankings and standings
- managing contests and problems

## Repository Structure

- `backend/` - Laravel-based API and judge integration
- `frontend/` - React user interface
- `judge-box/` - sandbox environment for code compilation and execution

## Getting Started

Refer to the individual `README.md` files in `backend/`, `frontend/`, and `judge-box/` for setup and deployment instructions.
